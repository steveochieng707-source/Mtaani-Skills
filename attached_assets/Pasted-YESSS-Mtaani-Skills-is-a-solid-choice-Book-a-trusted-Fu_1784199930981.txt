### *2. MVP Features - Version 1.0*
Build only these first. Don't add fluff.

| *For Customers* | *For Fundis/Students* | *For Admin/You* |
| --- | --- | --- |
| 1. Search by Skill + Location | 1. Sign up with ID + TVET Certificate | 1. Approve/Verify Fundis |

### *3. Tech Stack You Can Actually Build*
As a student, keep it simple:
1.  *Frontend*: React + Tailwind OR just HTML, CSS, JS if you’re starting
2.  *Backend*: Node.js + Express OR Laravel PHP OR Django Python
3.  *Database*: MySQL / PostgreSQL
4.  *Payments*: M-Pesa Daraja API for STK Push + B2C to pay fundis
5.  *Hosting*: Hostinger/Vercel + Free MySQL to start
6.  *Maps*: Google Maps API to show "Fundis near me"

### *4. Database Tables You’ll Need*
1.  users - id, name, phone, role: customer/fundi/admin
2.  fundis - user_id, skills, tvet_level, certificate_url, verified
3.  jobs - id, customer_id, fundi_id, description, location, price, status
4.  payments - job_id, amount, mpesa_code, status: escrow/released
5.  reviews - job_id, rating, comment

### *5. How You Make Money to Pay School Fees*
1.  *10% Commission*: Job is 2000 KES → You keep 200 KES, Fundi gets 1800 KES
2.  *Featured Fundi*: 500 KES/month to appear top of search
3.  *Urgent Job Boost*: Customer pays 100 KES extra to appear first
4.  *TVET Partnership*: Schools pay you to track student attachments

Math: 50 jobs/month x 200 KES commission = *10,000 KES*. 200 jobs = 40K
